# PromptWash UI Master Prompt v1.1

Design a modern prompt engineering workspace called PromptWash.

PromptWash helps users build, inspect, and evolve prompts.

---

# Core UX Moment

Messy thinking becomes structured prompts.

Users type rough ideas and the system progressively extracts:

Goal  
Audience  
Context  
Constraints  
Steps  
Output Format  
Tone  
Language  

The structure should appear while typing.

---

# Core Layout

Top Navigation

Three-column workspace:

Raw Input  
Structured Prompt  
Insights

Two drawers:

Variant Preview  
Execution Output

---

# Interaction Model

Live prompt analysis runs with small debounce.

Pipeline stages:

normalize → clean → analyze → lint → adapt

---

# Product Goal

Help users understand how prompts work while improving prompt quality.

PromptWash must feel like a prompt IDE rather than a prompt generator.

---

# Visual Direction

Dark glass interface.

Calm workspace.

Subtle animations.

Minimal cognitive overload.
