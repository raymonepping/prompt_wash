# PromptWash UI Tokens

This document defines design constants.

## Example Token Domains

- `color.background.primary`
- `color.panel.glass`
- `color.accent.primary`
- `color.accent.warning`
- `color.accent.success`
- `blur.panel`
- `radius.panel`
- `shadow.panel`

These tokens keep the UI visually consistent.
