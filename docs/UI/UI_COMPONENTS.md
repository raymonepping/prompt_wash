# PromptWash UI Components

This document defines reusable UI primitives.

It prevents the interface from turning into disconnected panels and one-off widgets.

## Layout Components

- AppShell
- TopBar
- WorkspaceLayout
- Panel
- SplitPane

## Workspace Components

- RawInputEditor
- StructuredPromptView
- InsightsPanel
- VariantTabs
- VariantPreview
- ExecutionPanel

## Data Components

- RunsTable
- ExperimentMatrix
- LineageGraph
- ModelLeaderboard
