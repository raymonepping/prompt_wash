# PromptWash Visual System

PromptWash uses a modern dark-first design system.

## Visual Tone

- Calm
- Precise
- Modern
- Minimal

Inspired by:

- Linear
- Arc Browser
- Raycast

## Color Palette

### Background Gradient

Deep navy to muted violet.

### Primary Accent

Blue.

### Insight Signals

Teal.

### Warnings

Amber.

### Errors

Red.

### Success

Green.

## Typography

### UI Text

- Inter
- Geist Sans

### Prompt Content

- JetBrains Mono
- Geist Mono

Minimum readable size: `13px`.

## Surfaces

Panels use subtle glass styling:

- Soft blur
- Rounded corners (`8px` to `12px`)
- Low shadow depth

## Animation

Animations must remain subtle.

Examples:

- Field emergence
- Highlight transitions
- Drawer slide-ins

Animations should feel calm rather than playful.

## Spacing

Large whitespace improves clarity.

Focus on readability and cognitive calm.

## Component Style

- Panels
- Cards
- Drawers
- Tabs
- Badges

All components should share consistent radius and spacing.
